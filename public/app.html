How to use IdeaGit

1. Enter your challenge and any constraints on the start page, then click Start Ideation.

2. Create your first idea manually or with AI using the chat panel on the right.

3. Chat to develop ideas. Send a message to modify the idea (creates a new node), get feedback, or ask questions. Modifications are saved as branches in the process tree on the left.

4. Navigate the process tree. Click any node to return to that version and continue from there — this creates a new branch.

5. Finalize ideas you are happy with. Finalized ideas appear in the middle panel.

6. Use + New Idea to start a completely separate idea for the same challenge.

7. Export CSV when done to save your full session.
