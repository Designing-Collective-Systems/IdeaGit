ABOUT IDEAGIT

Ideagit uses a process graph to help instructors observe and understand student-AI use. We are focused on researching the effectiveness of Ideagit in a classroom setting.

  • Start by uploading student data (CSV file)

  • Use the left side of the screen to quickly browse through different ideas

  • A node represents one version of an idea. Tags help indicate how much AI was used.

    User Created AI Generated Manual Edit AI Auto AI Chat Finalized

  • Drag to see all ideas on the graph. Use the new CSV button to upload a different data file.

  • Use the Questions button in the top right corner to access to the research questions once you are done looking at a student's work
